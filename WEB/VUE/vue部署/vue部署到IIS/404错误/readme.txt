错误现象描述:
1.Vue项目配置路由为history模式，浏览器F5刷新后，会显示404报错
2.或者路径带参数时,会显示404报错    例如：http://10.12.25.21:8111/?lineno=S14,S2



解决方式有2种:
方式1.安装URL Rewrite并且配置



方式2.增加配置文件web.config  (这种方式简单)
这种方式最简单
